Original microinteraction

Triggers

  • Triggered when you hover / hover off the Account Settings option block.

Rules

  • Pointer turns to standard hand icon.
  • Account Settings block moves up about 5-10 pixels.
  • On hover off, hand icon turns back to pointer.
  • Account Settings block moves down about 5-10 pixels.

Feedback

  • Pointer turns to standard hand icon.
  • Account Settings block moves up about 5-10 pixels.
  • On hover off, hand icon turns back to pointer.
  • Account Settings block moves down about 5-10 pixels.

Loops/Modes

  • There are no loops or modes for this microinteraction.

New microinteraction

Triggers

  • Triggered when you hover / hover off the Account Settings option block.

Rules

  • Pointer turns to standard hand icon.
  • Blue-purple border fades in around the block, growing from 1 - 5 pixel width in .4 seconds.
  • Gray sparkles will twinkle: The four lines of each will elongate from 0 length and shorten back to 0 length. Rightmost sparkle will activate .2 seconds after the other.
  • Two cogs rotate, yellow spins 180 degrees clockwise, purple cog spins 180 degrees counter-clockwise. The rotation speed starts fasts and slows down as the animation plays through the full .4 second length.
  • On hover off, cogs rotate another 180 degrees in the same direction as on hover, and border fades back to 0 width. No sparkles.

Feedback

  • Pointer turns to standard hand icon.
  • Blue-purple border fades in around the block, growing from 1 - 5 pixel width in .4 seconds.
  • Gray sparkles will twinkle: The four lines of each will elongate from 0 length and shorten back to 0 length. Rightmost sparkle will activate .2 seconds after the other.
  • Two cogs rotate, yellow spins 180 degrees clockwise, purple cog spins 180 degrees counter-clockwise. The rotation speed starts fasts and slows down as the animation plays through the full .4 second length.
  • On hover off, cogs rotate another 180 degrees in the same direction as on hover, and border fades back to 0 width. No sparkles.

Loops/Modes

  • There are no loops or modes for this microinteraction.
